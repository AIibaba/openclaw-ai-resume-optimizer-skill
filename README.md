*ATS-friendly resume optimization with keyword matching, achievement quantification, and customization.*

An intelligent resume optimization system that helps job seekers craft resumes that pass Applicant Tracking Systems and impress hiring managers. Analyzes job descriptions, matches keywords, quantifies achievements, and tailors your resume for maximum impact.

## Features

| Feature | Description |
|---------|-------------|
| ATS Compatibility Scanner | Analyzes resume format and content for ATS compatibility and suggests fixes |
| Keyword Matcher | Extracts key requirements from job descriptions and maps them to your experience |
| Achievement Quantifier | Transforms vague responsibilities into measurable, impact-driven bullet points |
| Resume Tailor | Customizes your resume for each specific job application automatically |
| Cover Letter Generator | Creates personalized cover letters that complement your optimized resume |
| Industry Benchmarking | Compares your resume against industry standards and top performers |

## Getting Started

```bash
# Install via OpenClaw
openclaw install AIibaba/openclaw-ai-resume-optimizer-skill
```

## Usage

```
optimize [resume]          # Full resume optimization analysis
match [resume] [job desc]  # Match resume to specific job posting
quantify [bullet points]   # Transform bullets into impact statements
tailor [resume] [job]      # Customize resume for a target role
cover letter [job desc]    # Generate a matching cover letter
```
